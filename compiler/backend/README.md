# Compiler Backend

Language: C++.

## Requirements

- Docker
- Linux or WSL (Windows)

## Build & Run

**Linux or WSL**:

Build:

```
chmod +x ./host/build.sh
./host/build.sh
```

Run:

```
chmod +x ./host/run.sh
./host/run.sh <path_to_module>
```

**Other platforms**

Good luck trying, it won't be possible.
